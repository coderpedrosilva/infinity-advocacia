# Infinity Advocacia — Site Institucional

> **⚠️ Aviso:** Este é um projeto fictício criado exclusivamente para fins de estudo e demonstração de design/desenvolvimento. O escritório **Infinity Advocacia não existe** — todos os dados (endereço, telefone, e-mail, advogados, depoimentos e artigos) são completamente fictícios.

Site institucional fictício desenvolvido a partir de um design criado no **Figma**, com o objetivo de praticar a conversão de layouts para código.

## Sobre o projeto

O layout foi desenhado no Figma e convertido para um site estático em HTML, CSS e JavaScript puro, mantendo fidelidade ao design original — cores, tipografia, espaçamentos e imagens.

## Tecnologias

- HTML5
- CSS3 (sem frameworks)
- JavaScript vanilla

## Design

- **Ferramenta:** Figma
- **Paleta de cores:** Navy `#132a58` · Dourado `#937531` · Slate `#4f6389`
- **Tipografia:** Roboto (Google Fonts)

## Seções do site

1. **Hero** — Chamada principal com foto de fundo e overlay dourado
2. **Quem Somos** — Apresentação do escritório
3. **Área de Atuação** — 4 cards com links para páginas detalhadas (Civil, Familiar, Imobiliário, Empresarial)
4. **Equipe** — Fotos e nomes dos advogados
5. **Depoimentos** — Avaliações de clientes em círculos sobrepostos
6. **Blog** — Artigos jurídicos com páginas completas
7. **Contato** — Formulário + dados de contato

## Como abrir

Basta abrir o arquivo `index.html` diretamente no navegador. Não requer servidor ou instalação.
